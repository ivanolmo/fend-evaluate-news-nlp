Udacity FEND NLP Project

This app utilizes the Aylien Text Analysis API to analyze and classify your input search term or URL. The API is a package of Natural Language Processing and Machine Learning-powered APIs for analyzing and extracting various kinds of information from textual content.

There are currently two search options. The first one uses Aylien's Sentiment Analysis. It will return information on your search term, such as the tone (positive, neutral, or negative), and whether or not the term is subjective or objective.

The second option uses Aylien's Summarization feature, and requires that you provide a URL. This option is an easy way to summarize a news article or blog post into a few key sentences. At this time, this option is limited to 4 summary sentences.
